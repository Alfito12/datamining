import pandas as pd
import matplotlib
import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np
import statistics as st
from scipy import stats

url="https://raw.githubusercontent.com/Alfito12/dataset/main/heart_disease_patients.csv"

c=pd.read_csv(url)

Pasien Penyakit Jantung

  1. age : Usia pasien

  2. sex : jenis kelamin

  3. cp : Jenis nyeri dada

  4. trestbps : Tekanan darah istirahat (dalam mm Hg saat masuk rumah sakit)

  5. chol : Kolesterol serum dalam mg/dl

  6. fbs : Gula darah puasa > 120 mg/dl (1 = benar; 0 = salah)

  7. restecg : Hasil elektrokardiografi istirahat

  8. thalach : Detak jantung maksimum tercapai

  9. exang : Angina yang diinduksi oleh olahraga (1 = ya; 0 = tidak)

  10. oldpeak : Depresi ST yang diinduksi oleh olahraga relatif terhadap istirahat

  11. slope : Kemiringan segmen ST latihan puncak

Rumus Mean#

Dalam statistika dan matematika, mean disebut juga dengan rata-rata. Jadi, definisi mean adalah nilai rata-rata dari sekumpulan data yang terdiri dari dua atau lebih.. Berikut rumus mean untuk data tunggal: $\(\bar{x}=\frac{\sum x_i}{n} \)$ Keterangan:

\(\bar{x}:\) nilai rata-rata

\(\mathrm{x}_{\mathrm{i}}\) : nilai data ke-i

n : banyaknya data

mean = {'age':np.mean(c['age']),
		'sex': np.mean(c['sex']),'cp':np.mean(c['cp']),
    'trestbps':np.mean(c['trestbps']),
    'chol':np.mean(c['chol']),'fbs':np.mean(c['fbs']),
    'restecg':np.mean(c['restecg']),'thalach':np.mean(c['thalach']),
    'exang':np.mean(c['exang']),'oldpeak':np.mean(c['oldpeak']),
    'slope':np.mean(c['slope'])
    }

df = pd.DataFrame(mean, index=['mean'])

Rumus Median#

Median disebut juga dengan nilai tengah. Median diperoleh dengan cara mengambil nilai tengah pada data yang telah disusun secara berurutan dari angka terkecil ke terbesar. Pada data tunggal untuk menentukan median yaitu: Misalkan banyaknya data n,

Untuk \(n\) ganjil: \(M e=x_{\frac{1}{2}(n+1)}\)

Untuk \(n\) genap: \(M e=\frac{\frac{x_n}{2}+\frac{x_n}{2}+1}{2}\)

Keterangan: \(x_{\frac{n}{2}}=\) data pada urutan ke- \(\frac{n}{2}\) setelah diurutkan.

median = {'age':np.median(c['age']),

Rumus Standar Deviasi#

Standar deviasi atau simpangan baku biasanya diajarkan pada ilmu statistik untuk mengukur tingkat kesamaan atau kedekatan dalam suatu kelompok.

Standar deviasi adalah nilai statistik yang sering kali dipakai dalam menentukan kedekatan sebaran data yang ada di dalam sampel dan seberapa dekat titik data individu dengan mean atau rata-rata nilai dari sampel itu sendiri.

Untuk Data Tunggal : \(S=\sqrt{\frac{\sum\left(x_i-\bar{x}\right)^2}{n}}\)

\(S=\) Standar deviasi

\(x_i=\) nilai \(\mathrm{x} \mathrm {ke-i}\)

\(\bar{x}=\) nilai rata-rata data

\(n=\) jumlah data

std = {'age':st.stdev(c['age']),
		'sex':st.stdev(c['sex']),'cp':st.stdev(c['cp']),
    'trestbps':st.stdev(c['trestbps']),'chol':st.stdev(c['chol']),
    'fbs':st.stdev(c['fbs']),'restecg':st.stdev(c['restecg']),
    'thalach':st.stdev(c['thalach']),'exang':st.stdev(c['exang']),
    'slope':st.stdev(c['slope'])
    }

RUMUS PERSENTIL DATA TUNGGAL $\( P_i=\text { data } k e-\frac{i(n+1)}{100} \)$

Jika persentil 25% maka: $\( P_i=\text { data } k e-\frac{25(n+1)}{100} \)$

Jika persentil 50% maka: $\( P_i=\text { data } k e-\frac{50(n+1)}{100} \)$

Jika persentil 75% maka: $\( P_i=\text { data } k e-\frac{75(n+1)}{100} \)$

Keterangan:

\(\mathrm{P}=\) persentil ke \(-\mathrm{i}\)

\[ i=1,2,3, \ldots, 99 \]

n = banyak data

percentile25 = {'age':np.percentile(c['age'],25),
